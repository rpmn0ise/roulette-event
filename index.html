ℹ Comment fonctionne le tirage ?

Le tirage utilise Math.random(), le générateur aléatoire natif du navigateur. Chaque option a exactement la même probabilité de sortir, quelle que soit sa position sur la roue.

Tout le calcul se fait dans ce fichier, côté client : aucun serveur externe, aucune donnée envoyée nulle part. Tu peux vérifier toi-même en ouvrant le code source (clic droit → « Afficher le code source de la page », ou F12).

Le lien de partage encode uniquement le titre, les options et les réglages — jamais un résultat pré-calculé. Chaque personne qui ouvre le lien et clique sur la roue obtient un tirage indépendant.
